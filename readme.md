# TuristGo

TuristGo es un sistema web orientado a la gestión de reservas de servicios de transporte turístico.

## Estado actual del proyecto
Fase de levantamiento de requerimientos, definición de alcance y documentación inicial.

## Objetivo inicial
Construir una primera versión funcional (MVP) que permita:
- mostrar servicios al cliente
- registrar reservas
- gestionar reservas internamente
- asignar servicios a conductores/guías
- administrar precios de forma manual

## Enfoque de trabajo
El proyecto se está desarrollando de forma incremental, priorizando:
1. entendimiento del negocio
2. documentación funcional
3. definición de alcance
4. validación con cliente
5. desarrollo técnico

## Documentación
La documentación principal se encuentra en la carpeta `docs/`.
