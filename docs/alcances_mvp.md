# Alcance MVP - TuristGo

## Objetivo del MVP
Construir una primera versión funcional del sistema que permita validar la operación digital básica del servicio principal de TuristGo.

## Qué sí incluye el MVP

### Sitio web público
- Página de inicio
- Presentación del servicio principal
- Información básica del servicio
- Botón para reservar

### Módulo de reserva
- Formulario de reserva
- Registro de datos del pasajero
- Registro de datos del vuelo
- Registro de cantidad de pasajeros
- Registro del servicio solicitado
- Registro del precio aplicado al momento de la reserva

### Panel administrativo
- Visualización de reservas
- Consulta de detalle de cada reserva
- Cambio de estado de la reserva
- Asignación manual de conductor/guía
- Configuración manual de precios

## Qué no incluye el MVP

- Pago online
- App para conductores
- Seguimiento automático de vuelos
- Marketplace de guías externos
- Cobertura nacional completa
- Automatización avanzada de precios
- Emisión automática de boletas
- Integraciones externas complejas

## Cobertura del MVP
- Servicio principal: aeropuerto → alojamiento
- Región inicial: Región Metropolitana

## Estados iniciales propuestos para la reserva
- Pendiente
- Confirmada
- Asignada
- Completada
- Cancelada

## Supuestos del MVP
- El equipo interno revisará reservas manualmente.
- La disponibilidad de conductores se controlará manualmente.
- El precio se administrará desde panel.
- Los cambios de reserva serán gestionados manualmente.
