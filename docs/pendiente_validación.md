# Pendientes de validación - TuristGo

## Temas pendientes con cliente

1. ¿La reserva se confirma manual o automáticamente?
2. ¿Qué pasa si no hay conductor disponible?
3. ¿Qué pasa si un vuelo se retrasa y genera conflicto con otra reserva?
4. ¿Hasta cuándo se puede modificar una reserva?
5. ¿Hasta cuándo se puede cancelar una reserva?
6. ¿Se enviará algún tipo de confirmación al cliente?
7. ¿El cliente reservará sin cuenta o con cuenta?
8. ¿Cómo esperan gestionar pagos en una fase futura?
9. ¿Cómo operan actualmente la asignación interna de servicios?
10. ¿Qué campos adicionales desean guardar en una reserva?

## Nota
El proyecto puede avanzar a nivel documental y luego técnico inicial, pero estos puntos deben validarse antes de cerrar completamente la lógica del sistema.
