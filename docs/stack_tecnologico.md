# Stack tecnológico - TuristGo

## Objetivo
Definir las tecnologías a utilizar para el desarrollo del MVP del sistema.

---

## Backend
- Lenguaje: Python
- Framework: FastAPI
- Validación de datos: Pydantic
- ORM (futuro): SQLAlchemy

---

## Base de datos
- PostgreSQL

---

## Frontend
- HTML
- CSS
- JavaScript

Se prioriza simplicidad para no bloquear el desarrollo del MVP.

---

## Control de versiones
- Git
- GitHub

---

## Enfoque
Se prioriza:
- rapidez de desarrollo
- claridad en la arquitectura
- facilidad de mantenimiento
- escalabilidad futura

---

## Notas
- El sistema podrá evolucionar a tecnologías más complejas en fases posteriores.
- Se evita sobre-ingeniería en el MVP.
